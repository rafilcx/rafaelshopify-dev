# DESIGN.md — Rafael Silva / Tittanium Personal Landing Page

## Brand Context

Rafael Silva is a Shopify Developer & E-commerce Systems Builder.

He builds digital systems for e-commerce by connecting Shopify, AI, dashboards, automation, data and Creative Performance.

Rafael is a former designer who evolved into a hybrid builder across:
- Shopify theme development
- UX/UI and conversion-focused interfaces
- dashboards and data intelligence
- AI systems and automation
- operational knowledge bases
- Creative Performance workflows
- e-commerce systems architecture

The page should not look like a generic portfolio.

It should feel like a premium, intelligent and motion-first operating system for modern e-commerce growth.

The core positioning is:

> Shopify, IA & Sistemas para E-commerce

Main message:

> Construo sistemas digitais para e-commerces que precisam conectar loja, dados, automações, dashboards e creative performance em uma operação mais inteligente.

The page must make people understand in 3 seconds:

Rafael helps e-commerce brands connect store, data, AI, dashboards, automation and creative workflows into clearer, faster and more measurable systems.

It should feel like an operating system running in real time, not a flashy animation demo. Motion must be elegant, purposeful, fast enough to feel responsive, and never childish or distracting.

Motion should feel:
- refined
- intelligent
- system-like
- subtle but alive

- purposeful
- responsive
- elegant

- chaotic motion
- heavy cyberpunk animation
- childish effects
- slow heavy transitions
- animation that hurts readability

- prefers-reduced-motion
- accessibility
- performance
- mobile usability
